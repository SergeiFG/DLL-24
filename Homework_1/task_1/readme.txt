В данном задании реализована линейная регрессия через 1 нейрон меодуля pytorch.
Перед началом нахождения коэффициентов регрессии исходый датасет загружается, очищается от строк, имеющих значния NaN, затем данные перемешиваются, так как в исходом датасете они сильно сгруппированны по близости океана. После этого числовые данные нормируются до интервала [-1; 1], а данные о близости океана (представляюще собой 5 состояний) заменяются на таблицу из 0 и 1, в которой 1 отвечает за соответствие каждому состоянию.
80% данных использованы для обучающей выборки, 20% - для тестовой.
Использован оптимайзер SGD и MSE loss функция. Для оценки результатов обучения строятся графики loss функции на тренировочной (loss value) и тестовой (accuracy) выборке.
